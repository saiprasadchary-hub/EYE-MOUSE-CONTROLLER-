✅ 1. Python Extension for VS Code
VS Code Extension: Install the official Python extension by Microsoft from the Extensions Marketplace.
This enables Python linting, debugging, and execution inside VS Code.
✅ 2. Required Python Libraries
Install the following packages via pip:
pip install opencv-python mediapipe pyautogui
Explanation:
opencv-python: For video capture and image processing.

mediapipe: For face and eye landmark detection.

pyautogui: For controlling the mouse pointer and simulating clicks.

✅ 3. Optional (but helpful) Python Tools
You might also want:
pip install numpy
